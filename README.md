## 🔍 Overview
Smart-Lender is a machine learning-based loan prediction system that helps financial institutions make data-driven decisions. The system uses a Random Forest classifier trained on historical loan data to predict whether a loan application should be approved or rejected.

## ✨ Features
- **Instant Predictions**: Get loan approval status in seconds
- **User-Friendly Interface**: Clean and intuitive web interface
- **Probability Scores**: View approval and rejection probabilities
- **Comprehensive Input**: Considers 11 different factors for prediction
- **REST API**: Programmatic access to prediction capabilities
- **Visual Analytics**: Data visualizations for better understanding

## 🛠️ Tech Stack
- **Backend**: Python, Flask
- **Machine Learning**: scikit-learn (Random Forest)
- **Data Processing**: Pandas, NumPy
- **Frontend**: HTML, CSS, JavaScript
- **Visualization**: Matplotlib, Seaborn
